# solid-components
using the simplycode architecture, a way to make an interface for use cases for the "everything app" possibilities.


# design components
* git
  * commits the current app configuration to a git repository with UI buttons

* kanban
  * make todo lists and link them to chat and other items, automate flow
 
* chat
  * integrated chat modules to kanban

* planning
  * personal planning overview of team members: what and when they do things on the kanban, links back to the kanban boards.
